# APB - Test 🎓🌍

**APB - Test** є тестовим завданням від компанії APB.

## 🌐 Додаток доступний за посиланням:

👉 [apb-test.vercel.app](https://apb-test.vercel.app)

## 📦 Локальний запуск

1. Клонувати репозиторій:

```bash
https://github.com/Mykhailenko-Oleksandr/apb-test.git
```

2. Перейти в склонований проєкт і встановити залежності:

```bash
npm install
```

3. Запустити локальний сервер:

```bash
npm run dev
```

## 📝 Опис функціоналу

- Головна сторінка з навігацією між основними розділами.
- Список змінних (отримання даних з NHTSA).
- Розшифровка автомобільних VIN-кодів та відображення
  отриманих даних.
- Сторінка за всіма доступними змінними.
- Детальна сторінка змінної з повною інформацією (Name, GroupName, Description).
- Використання React Router для маршрутизації та React Query для роботи з API.
